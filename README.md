# TaskFlow 🗂️

## Descripción
TaskFlow es una aplicación de consola desarrollada en C# como parte del proyecto
de NovaTech. El sistema está orientado a la gestión de tareas, permitiendo a los
usuarios organizar, visualizar y administrar sus actividades desde la línea de comandos.

> Proyecto desarrollado en el marco de la materia Programación Aplicada II —
> ESCMB, Universidad Nacional de Córdoba.

---

## 🚀 Cómo ejecutarlo

### Requisitos previos
- Tener instalado [.NET SDK](https://dotnet.microsoft.com/download) (versión 6.0 o superior)
- Tener instalado [Visual Studio 2022](https://visualstudio.microsoft.com/) o Visual Studio Code
- Tener instalado [Git](https://git-scm.com/)

### Pasos

**Opción A — Visual Studio**
1. Clonar el repositorio:
```bash
   git clone https://github.com/TU_USUARIO/taskflow.git
```
2. Abrir el archivo `First.sln` con Visual Studio
3. Presionar `F5` o el botón **Ejecutar** para compilar y correr el proyecto

**Opción B — Terminal / CMD**
1. Clonar el repositorio:
```bash
   git clone https://github.com/TU_USUARIO/taskflow.git
```
2. Navegar a la carpeta del proyecto:
```bash
   cd taskflow/First
```
3. Compilar y ejecutar:
```bash
   dotnet run
```
